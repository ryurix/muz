Номер

Порядковый номер правила для запуска.

Каталог

Каталог для которого срабатывает правило (так же во всех подкаталогах).

Группа

Если указано — Правило сработает только товаров из выбранной группы.

Производители

Если указано — Рассматриваются только товары с данным постащиком и будет проверяться наличие у поставщика.

Наличие

Если указано "В наличии" — Рассматриваются товары с наличием хотя бы в одной актуальной синхронизации.

Если указано "Отсутствуют" — Рассматриваются только товары без синхронизаций или с устаревшими синхронизациями или если по всем синхронизациям нули.

Минимальная/Максимальная цена

Если стоит больше нуля — в правиле обрабатываются только товары с розничной ценой меньше или больше указанной.

Взять цену

Это основной параметр определяющий работу правила.

Розничную поставщика — будет проставлена розничная цена из прайса (синхронизации) указанного поставщика.

Оптовую поставщика — будет проставлена оптовая цена из прайса (синхронизации) указанного поставщика.

Минимальную / Среднюю / Максимальную — будет вычислена соответствующая цена из синхронизаций. Если указан параметр "В наличии" — будут рассматриваться только синхронизации в которых есть наличие.

Обнулить всех — товарам соответствующим этому правилу будет установлена цена ноль.

Коррекция цены

Итоговая цена товара будет уменьшена/увеличена на указанный процент.

Финальное правило

Если данное правило было применено к товару — следующие правила уже не будут на него действовать.
